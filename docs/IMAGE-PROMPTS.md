# Skola image generation prompt pack

Every image on the site is served from committed files under `public/`. Random
Picsum/Pravatar is gone — each picture is meant to match its topic and to feel
**Nigerian**, not generic Western stock.

Right now those paths hold on-brand teal **placeholders** (generated by
`scripts/gen-placeholders.mjs`). To ship the real look, generate each image with
the prompt below and **save it over the placeholder using the exact same
filename**. No code changes are needed — the site already points at these paths.

## House style (paste into every prompt)

> Authentic, documentary-style Nigerian photography. Real Nigerian people and
> settings (Lagos, Ibadan, Kano, Enugu, Zaria, Nsukka, etc.). Natural warm
> daylight, true-to-life skin tones, candid and un-staged, not a glossy stock
> photoshoot. No text, no watermarks, no logos. Nobody staring blankly at the
> camera unless it is a portrait.

**Negative / avoid:** foreign/Western campuses, all-white or non-African cast,
snowy or European scenery, cartoon or 3D render look, text overlays, watermarks.

**Format & delivery**

| Set | Path | Aspect | Suggested px |
| --- | --- | --- | --- |
| Category tiles | `public/categories/<slug>.jpg` | 5:4 portraitish | 1000×800 |
| Resource covers | `public/catalog/<id>.jpg` | 16:10 landscape | 1200×750 |
| Avatars | `public/avatars/f1..f8, m1..m8.jpg` | 1:1 | 600×600 |
| Scenes | `public/scenes/campus.jpg`, `library.jpg` | portrait | 1100×1500 |

Export as JPG, quality ~80. Keep filenames exactly as listed.

After replacing the files: `git add public && git commit && push`, then **re-seed
the production database** so browse/search reflect the new catalog rows too
(`npm run db:seed` against the prod `DATABASE_URL` — it is non-destructive).

## Avatar pool (`public/avatars/`)

16 head-and-shoulders **portraits** of different Nigerian people, plain neutral
background, friendly, looking toward camera, realistic (not glossy). Vary
ethnicity (Yoruba, Igbo, Hausa/Fulani, South-South), age and dress; include a
couple of older faces for the "lecturer" testimonials, and one or two women in a
hijab. The site assigns each testimonial/review name a face by gender, so keep
the split below.

**Women — `f1.jpg` … `f8.jpg`:** 8 distinct Nigerian women. Suggested mix: two
early-20s students, two mid-20s, one in a hijab, one older woman (40s–50s, a
lecturer), the rest varied.

**Men — `m1.jpg` … `m8.jpg`:** 8 distinct Nigerian men. Suggested mix: three
students (early-20s), one in northern dress/kaftan, one older man (40s–50s,
a lecturer/supervisor), the rest varied.

Per portrait prompt template:

> Head-and-shoulders portrait of a [age] Nigerian [woman/man], [feature/dress
> detail], plain soft neutral background, natural warm light, friendly relaxed
> expression, realistic documentary photography, no text.
